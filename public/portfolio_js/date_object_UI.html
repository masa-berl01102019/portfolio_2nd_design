Description

The maximum week of the calendar is 6 weeks, and a week is consist of 7 days. Therefore, I created an array to store the numbers from 1 to 42 and considered day of the week from sunday to saturday as the numbers from 0 to 6. Secondary, I obtained day of the week of the first day of month, and the last day of month. Then, use the map function to subtract the value returned by the getDate function from each value in the array. Finally, I created dynamic elements and inserted it from 1 to the last day of month using conditional branch to assign blanks to exceptions.
