This structure of mini game is below.
The words are stored in an array and shuffled by using Fisher–Yates shuffle algorithm in order to create random unbiased permutation so that the same word is not repeated in a single play. Each single character which user typed is taken from the event object of keydown and is validated every nth String object (the displayed word) in order from the top. when it's correct, it's replaced in '_' , and move to the next character. After the game, it shows the result that how much correctly user was able to type the words within the limited time by showing accuracy rate.

    This structure of mini game is below.

It's created dynamically elements by constructor function when its class instance which is controlling panels is created ( Board class ). Those elements creating panels has been stored in an array.
